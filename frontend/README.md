# Frontend

Vite + React + TypeScript app with shadcn/ui and Tailwind.

## Commands

- `npm run dev` – start dev server (port 8080)
- `npm run build` – production build
- `npm run preview` – preview production build
- `npm run lint` – ESLint
- `npm run test` – Vitest

## API

When you add a backend, set the API base URL via environment variables (e.g. `VITE_API_URL`) and use it in your API client or TanStack Query configuration.
