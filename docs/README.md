## Quick links

- **Run frontend:** `npm run dev` (from repo root) → http://localhost:8080  
- **Run backend:** `npm run dev:backend` → http://localhost:3000  
- **Repo structure:** `frontend/` (Vite + React), `backend/` (Express + TypeScript)
